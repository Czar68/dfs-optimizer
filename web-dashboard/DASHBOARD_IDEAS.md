## Future ideas

- **Export visible table** — Button to download current view (cards or top legs) as CSV
- **Keyboard shortcut** — e.g. Ctrl+R to refocus / refresh hint
- **Dark/light theme toggle** — Optional preference
- **Compare to last run** — Show diff (new/removed cards) when manifest timestamp changes
- **Time-window filter** — e.g. “Games starting in next 2h” using leg gameTime
- **Favorites / pins** — Mark cards to keep at top or in a separate list
- **Leg-level copy from Top Legs** — Click row to copy "Player STAT oX.X"
- **Mobile layout** — Stack metrics below player on small screens
- **Tooltips** — Hover on Score/EV/Win% for short formula or definition
- **Sound or badge** — Optional alert when new run is detected (e.g. manifest timestamp change)
