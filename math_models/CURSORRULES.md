# Math Models — Lockdown Policy

**All modules in this directory contain production DFS mathematical formulas and logic.**

- **Do NOT modify any formula, constant, or computational logic** in any file under `math_models/` without a **formal peer-review/approval** step.
- Allowed without approval: typo fixes, comments, type annotations, and re-exports. Any change to equations, thresholds, payout usage, or stake/Kelly logic **requires** explicit approval before editing.
- Each module has a corresponding `.cursorrules` file that enforces this for that file.
